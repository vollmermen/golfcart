* Anforderungen

ANF001 Erzeuge einen Puls jedes Mal wenn ein Punkt auf dem Rad an einem festgelegten Stelle am Fahrzeug vorbeikommt
ANF002 Erzeuge einen Puls jedes Mal wenn ein Benutzer den blauen Taster des Evaluation-Boards drückt
ANF003 Stelle den Zeitabstand zwischen zwei aufeinander folgenden Pulsen fest
ANF004 Errechne aus dem Zeitabstand zwischen zwei aufeinander folgenden Pulsen die Fahrzeuggeschwindigkeit

* Anleitung

Eine Änderung erstellen:
1. im (lokalen) Arbeitsverzeichnis die Datei(en) ändern
2. git add <dateiname> : Änderungen auf die (lokale) Staging Area bringen ("auf die Bühne bringen")
3. git commit <dateiname> : Änderungen in die (lokale) Datenbank schreiben
4. git push origin master : Änderungen von der (lokalen) Datenbank auf das (remote) Verzeichnis bei github.com schreiben
5. Hier kann es sein, dass jemand Anderes vor Dir schon Änderungen gemacht hat, die musst Du holen (fetch) und dann mit Deinen Änderungen zusammenführen (merge)
